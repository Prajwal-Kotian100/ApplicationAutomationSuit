# Application Automation Suite

This project contains an automation testing framework built using Selenium WebDriver with Java.

## 🚀 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Postman (API Testing)

## 📌 Features
- End-to-end UI automation
- API testing validation
- Page Object Model (POM) design
- Reusable and scalable framework

## 📂 Project Structure
- test cases
- page objects
- utilities
- test data

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ/Eclipse
3. Run using TestNG XML or Maven
